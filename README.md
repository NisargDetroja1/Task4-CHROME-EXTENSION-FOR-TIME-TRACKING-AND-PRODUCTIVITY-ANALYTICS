# 📌 Project Description

An Advanced Chrome Extension that tracks time spent on websites and provides productivity analytics using Chrome APIs and JavaScript.

The extension monitors browsing activity, calculates productivity score, tracks total browsing time, and displays website usage analytics in a modern dashboard interface.

---

# 🚀 Technologies Used

* HTML
* CSS
* JavaScript
* Chrome Extension APIs

---

# ✨ Features

* Real-time website time tracking
* Productivity analytics dashboard
* Productivity score calculation
* Productive vs non-productive websites
* Total browsing time tracking
* Total websites visited counter
* Animated progress bars
* Modern dark UI design
* Glassmorphism interface
* Local storage support

# ⚙️ How It Works

1. The extension tracks active browser tabs
2. Stores website usage time using Chrome Storage API
3. Calculates productivity score
4. Displays analytics in popup dashboard
5. Updates website statistics automatically
